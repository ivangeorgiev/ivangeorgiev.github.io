Both Python 2 and Python 3 provide HTTP server libaries that allow you to start HTTP server directly without any programming. This feature is very useful when workin on a web application that requires a web server.

The HTTP server modules in Python 2 and Python 3 are different.

Python 2

# Start Python HTTP Server
$ python -m SimpleHTTPServer 80

For more information check the Python 2 SimpleHTTPServer documentation page.

Python 3

# Start Python 3 HTTP Server
$ python -m http.server 80

For more information check the Python 3 http.server documentation page.
